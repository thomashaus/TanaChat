# Archived Specifications

This directory contains completed specifications for TanaChat features that have been fully implemented and deployed.

## Archived Features

| Specification | Version | Completed | Description |
|---------------|---------|-----------|-------------|
| [Outline Generation](./outline-generation.md) | 1.0 | 2025-12-10 | Generate hierarchical outlines from Tana JSON exports |

## Archive Process

Specifications are moved to this archive when:

1. **Implementation Complete**: All features are implemented and tested
2. **Deployed**: Feature is running in production
3. **Documentation Updated**: User documentation is complete
4. **Post-Deployment**: Feature has been stable for at least 1 week

## Accessing Archived Specs

While specifications are archived, they remain valuable for:

- **Reference**: Understanding existing implementations
- **Maintenance**: Bug fixes and updates
- **Similar Features**: Patterns and approaches for new features
- **Audit Trail**: Historical record of decisions

## Maintenance

Archived specifications may still be updated for:

- **Bug Fixes**: Correcting errors in the specification
- **Version Updates**: When breaking changes are made
- **Security Updates**: When security considerations change

For changes to archived specifications:
1. Create a new version in the archive
2. Update the version history
3. Notify the development team
4. Update any dependent documentation
