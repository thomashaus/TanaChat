# Web Interface Overview

The TanaChat frontend is a modern React application built with Vite and Tailwind CSS.

## Features

- **Chat Interface**: Interact with your Tana workspace via AI.
- **File Manager**: Upload and manage Tana exports.
- **Visualizations**: View knowledge graphs of your notes.

## Development

Run the development server:

```bash
cd app
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.

## Architecture (Frontend)

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State Management**: React Hooks / Context
- **Routing**: React Router
